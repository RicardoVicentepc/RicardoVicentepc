<h1 align="center">Olá 👋, sou Ricardo Vicente</h1>

  > Atualmente sou estudante de programação, ingressado na faculdade de tecnologia sptech, no curso de Análise e Desenvolvimento de Sistemas. Sempre estou em busca de problemas no meu cotidiano, para resolver com programação. Mas, claro, nem tudo nessa vida se resolve com programação, seria muito fácil não acha? O legal do problema é saber quebrar a cabeça e estruturar ele para que outros possam entender sua resolução. Ou seja, um projeto bom é aquele com uma ótima documentação.

- 🦅 Em meados de 2021 a 2022, fui uns dos 7 desenvolvedores responsáveis ​​para resolução de problemas em filas de restaurante. Claro, isso é impossível reduzir totalmente, mas, desenvolver este projeto para quando um restaurante estava cheio, ele ia recomendar o cliente para outro parecido. Mas isso não iria acontecer por generosidade, o restaurante não iria abrir vaga para concorrência. Então para resolver isso, o restaurante ganharia uma porcentagem do que o cliente consumiria no outro restaurante indicado. [NextMeal](https://github.com/Ancient-softwares)

- 🖥️ Me formei em 2022 como técnico de informática na Etec Guaianases. **Desenvolvimento de Sistemas.**

- 💉 No começo de 2023, realizei um projeto com 6 pessoas, com intuito de reduzir o desperdício de vacinas mal monitoradas, ou seja, vacinas que não foram refrigeradas corretamente. Realizamos o projeto com sensores de presença e temperatura, para reduzir as perdas de vácinas. [VacSense](https://github.com/MonitorHealth)

- 🍅 No final do primeiro semestre realizei um projeto desafiador com mais de 5 pessoas. Nosso objetivo era reduzir as perdas de tomates mal monitorados durante o trajeto. Estamos utilizando sensores de temperatura e umidade para saber o estado do tomate, e entregar em bom estado. [Pomodoro](https://github.com/SealSolution)

- 🍿 Atualmente estou desenvolvendo um projeto para monitorar os totens de segurança de cinema, utilizando Java como linguagem forte para realização deste mesmo. Pois sistemas de totens travam muito e necessitam ser monitorados para uso adequado.

- 🖥️ Atualmente estou concluindo minha Graduação na universidade Sptech.. **Análise e Desenvolvimento de Sistemas.**
